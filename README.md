# FitnessCenterApp

Bu proje Web Programlama dersi kapsamında geliştirilmiş bir spor salonu yönetim ve randevu sistemidir.  
ASP.NET Core MVC yapısı kullanılarak oluşturulmuştur. Projede salon yönetimi, eğitmen yönetimi, üyeler için randevu alma, API ile veri çekme ve basit bir yapay zekâ öneri sistemi yer alacaktır.
